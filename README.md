# AutoExpEnable
A 1.12.1 Vanilla WoW addon that notice player exp disabled.

Install location should be: ..\Interface\AddOns\AutoExpEnable

**Make sure you remove the "-main" from the "AutoExpEnable-main" folder so it becomes just "AutoExpEnable"**

当经验被关闭 在屏幕上方和聊天框里提示警告信息

可以在聊天框里输入`.xp on`开启经验获取，也可以右键头像让获取经验开关打开。
